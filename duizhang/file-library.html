易乐对账逻辑说明

适用范围本页逻辑仅用于易乐对账；后续新增的其他供应商对账页面会使用独立逻辑，不共用本页核对规则。
文件槽位1为旺店通代发表，2为运营登记表，3为易乐对账表，4为备用槽位；一键生成表只读取前3个已应用文件。
判断类型B列为整车购买或配件购买时进入核对；其他类型不参与查找，直接把B列内容写入W列。
查找顺序先用C列子客户同时查运营登记表和旺店通代发表；未形成已核实时，再用G列快递单号按同样范围查找。
基础结果同一个查找字段在运营登记表和旺店通代发表都命中，W列为已核实；只命中一边为待核实；都未命中为无记录。
退货/未发命中行出现红字、取消、退款、退货或未发时，X列写入退货/未发，并计入退货数量。
复核列位生成表会新增Y列复核结果、Z列异常类型、AA列复核说明；W/X保留基础核对结果。
类型/型号复核整车购买只应匹配运营登记表整车发货范围，配件购买只应匹配配件表；同时比对可识别的型号、规格、商品、SKU等字段。
数量复核按同一子客户优先汇总易乐行数、数量和快递单号，再与运营登记表命中行的要得数量、发货数量或件数对比，识别疑似多发或数量不一致。
取消发货复核运营登记表已取消、退款、退货、未发或红字，但易乐仍有快递单号或旺店通发货线索时，标记运营已取消但易乐未取消发货。
